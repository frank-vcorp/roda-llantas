# Carpeta de Interconsultas (Dictámenes de DEBY)

Esta carpeta contiene los dictámenes técnicos generados por DEBY para cada fix.

Cada dictamen sigue el formato `DICTAMEN_FIX-YYYYMMDD-NN.md` y documenta:
- Análisis de causa raíz
- Justificación de la solución
- Instrucciones de handoff

Ver `meta/SISTEMA-IDS.md` para el sistema completo de IDs.
